About the Bot

altego is a simple Discord bot created for learning, testing, and demonstrating Discord application features such as slash commands and interactions.

Features

  • Responds to basic slash commands
  • Simple greeting responses like Hello / Hi
  • Used for Discord bot development practice
  • Complies with Discord Developer requirements

Data & Privacy

The altego bot does not store personal user data. Any information processed is used only to respond to commands and is not saved.

Purpose

This bot is intended for educational use, testing Discord integrations, and understanding how Discord applications and interactions work.
